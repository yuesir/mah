# Mah 文档

Mah 是一款开源 HTML5 麻将接龙（Mahjong Solitaire / 上海麻将）游戏，作者 ffalt，MIT 许可证，当前版本 v1.19.3。支持 Web、桌面（Tauri）和 Android。

> ⚠️ 这不是中国传统 4 人麻将，而是单人配对消除类游戏。

## 一句话总览

- **架构**：Angular 22 standalone + 可变 Model 对象（无 NgRx）+ Web Worker 求解 + Tauri 多端
- **核心循环**：在多层 3D 牌阵中找出"上方无压、左右不同时夹"的成对牌进行消除，清空即胜
- **特色**：84 内置棋盘 + 种子化随机布局 + 保证可解的生成器 + 84+ 种子布局模式 + 37 语言
