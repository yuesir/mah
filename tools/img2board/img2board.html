Mah 图片转关卡 PNG alpha → .mah 布局文件

拖入一张图片，自动按 alpha 透明通道抠出前景轮廓，映射成单层关卡。下载 .mah 文件后拖进 Mah 编辑器的 Import 区域即可载入。 注意：图片转换出来是单层（z=0），几乎一定 Solved: 0，导入后需用编辑器复制层 + 删外圈造阶梯，或开始游戏时选 Solvable 生成器。
